# 📚 Spring Boot Book Repository API

A simple CRUD REST API built using **Spring Boot** and **MySQL** to manage a collection of books.

---

## 🚀 Features

- Add a new book
- Get all books
- Get book by ID
- Update a book
- Delete a book

---

## 🧰 Tech Stack

- 💻 JDK 21
- ⚙️ Spring Boot
- 🗃️ MySQL Database
- 🛠️ Maven
- 🔗 JPA + Hibernate
